选择器权重

  1. !important > 行间样式 > id > class/属性 > 标签选择器 > 通配符
    !important写在css某一属性后面
  2. CSS选择器 权重值(进制为256)
    !important 正无穷（正无穷+1>正无穷）
    行间样式 1000
    id 100
    class|属性|伪类 10
    标签|伪元素 1
    通配符 0

CSS

  • css参考手册（转w3c）： http://www.w3school.com.cn/cssref/index.asp
  • 属性 含义 默认值 属性值
    font-size 字体大小 16px 设置的是字体的高度，而非宽度
    font-weight 字体粗细 normal 细→粗：lighter，normal(相当于400)，bold(相当于700)，bolder（或者100,200,......,900）
    font-style 字体风格（斜体） normal normal正常，italic斜体，oblique倾斜（与italic看起来没啥区别）
    font-family 字体（宋体，黑体...） 取决于浏览器 normal正常，italic斜体，oblique倾斜（与italic看起来没啥区别）
    color 字体颜色
    1. 土鳖式（纯英文单词red,green）
    2. 颜色代码：#ff4400可简写为#f40
    3. 颜色函数：rgb(255, 255, 255)
    border 加个边框,用于画三角形 此属性有3个参数，更多参数详见w3c
    1. border-width
    2. border-style：设置线条样式
    3. border-color
    border-redius 圆角 正圆：50%
    text-align 水平对齐方式 取决于浏览器 left，center，right
    line-height 单行文本所占高度
    垂直居中（单行）：将单行高度与容器高度设成一致的（line-height == height）
    text-indent 首行缩进 2em：缩进两个文本距离（em可以用于指定倍数行高，如line-height:1.2em）
    position 定位 absolute、relative、fixed
    z-index 设置元素在第几层 0 必须有position才生效
    值越大，越靠近屏幕外
  • 此三角是用border画的

标签归类

  • 分类 特点 示例 css中display的值（可手工更改）

    不可用通过css改变宽高    		 span,strong,em,a,del inline

盒子模型

  • 外边距：margin（上-右-下-左，上-左右-下，上下-左右）
  • 盒子壁：border
  • 内边距：padding（上-右-下-左，上-左右-下，上下-左右）
  • 盒子内容：width+height
  • box

层模型（position）

  1. absolute：脱离原来位置进行定位（上升到新的一个图层）
    相对于最近的有定位的父级进行定位，如果没有则相对于文档进行定位（课时6-110分钟处）
  2. relative：保留原来位置进行定位（上升到新的一个图层）
  3. fixed：广告定位
