html标签及属性

  1. lang="en"告诉搜索引擎爬虫，我们的网站是关于什么内容的
  2. p段落标签，独成一段
  3. h1~h6标题标签，独成一段，加粗
  4. 4.strong加粗，em斜体，del删除
  5. div独占一行（section与div用法一致，更语义化），span无效果，容器，分块代码
  6. 回车、空格代表文本分隔符，多个等于一个
  7. 回车：<br>
  8. 空格：" " " "

引入CSS

  1. 行间样式
    div里加style属性
  2. 页面级css
    在head里写一个style
  3. 外部css引入
    在head里写一个link

选择器权重

  1. !important > 行间样式 > id > class/属性 > 标签选择器 > 通配符
    !important写在css某一属性后面
  2. CSS选择器 权重值(进制为256)
    !important 正无穷（正无穷+1>正无穷）
    行间样式 1000
    id 100
    class|属性|伪类 10
    标签|伪元素 1
    通配符 0

选择器（权重相同取后者）

  1. 标签选择器
  2. 属性选择器： id选择器 == 属性选择器 ?????
  3. 父子选择器/派生选择器： 自右向左 识别标签（浏览器底层为提升效率）
  4. 直接子元素选择器 （非直接子元素则无样式）
  5. 并列选择器： CSS中多个限制条件间不加空格 （类似and逻辑）
  6. 分组选择器： 逗号分隔每个选择器
