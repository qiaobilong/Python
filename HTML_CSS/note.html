html标签及属性

  1. lang="en"告诉搜索引擎爬虫，我们的网站是关于什么内容的
  2. p段落标签，独成一段
  3. h1~h6标题标签，独成一段，加粗
  4. 4.strong加粗，em斜体，del删除
  5. div独占一行，span无效果，容器，分块代码
  6. 回车、空格代表文本分隔符，多个等于一个
  7. 回车：<br>
  8. 空格：" " " "

  2. 页面级css

  3. 外部css引入

选择器权重

  1. !important > 行间样式 > id > class/属性 > 标签选择器 > 通配符
    !important写在css某一属性后面
  2. CSS选择器 权重值
    !important 正无穷（正无穷+1>正无穷）
    行间样式 1000
    id 100
    class|属性|伪类 10
    标签|伪元素 1
    通配符 0
